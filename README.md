Gunakan id "kirman-scrool-fixed-top" agar javascript berpungsi, contoh:

&lt;header id=&quot;kirman-scrool-fixed-top&quot; class=&quot;?&quot;&gt;
Your-elment-here
&lt;/header&gt;
